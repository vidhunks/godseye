# GodsEye

## Enterprise AI Governance & Security Platform

GodsEye is a cloud-native platform for discovering, governing, and securing enterprise AI ecosystems.

It continuously maps AI agents, MCP servers, tools, models, data sources, users, and policies into a unified governance graph while performing security and risk analysis.

---

## Features

- MCP Server Discovery
- Governance Graph
- Risk Assessment
- Blast Radius Analysis
- AI Asset Inventory
- Continuous Monitoring
- Dashboard
- OPA Policy Generation
